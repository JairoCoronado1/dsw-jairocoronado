Mockup

Los mockup es el diseño o prototipo de los diferentes sitios web que permite interactuar con el cliente,
y aprender de lo que desea que su sitio web esto establece una relacion directa cliente o empresario,
con el diseñador web.

 

en la cual permite que el cliente conozca como quedara su sitio web final que le permitira que los cibernautas,
puedan visitar su sitio web con los diferentes fines sean comercial, social, gubernamental, fabricación entre otras.
